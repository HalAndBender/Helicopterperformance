Instructions:
The app is intented to be used for pilots flying the AW139 helicopter under normal operating conditions near sea level.
It was created with reference to: Agusta Westland AW139 Rotorcraft Flying Manual, rev.20, pg. 9-126, "Hover Ceiling out of ground effect 2.5 minute power OEI".

Accurate calculation can only be provided for the following input ranges:
Dry Operating Mass(4400, 5500)
Fuel at site( 0, 1200)
Hover Height above MSL( 0, 500)
QNH( 970, 1050)
Temperature( -10, 40)
Headwind( 0, 40)
Performance Benefit( 0, 100)

After submission, please allow about 10 to 15 seconds for the results to be rendered.
A table with the results will be printed and reference lines (red) will be drawn on top of the performance chart provided by the manufacturer.
